### Инструкция

_все команды выполнять в терминале_


1. Склонируйте репозиторий
   ```sh
   git clone https://github.com/Xammirz/took_tz
   ```
2. Скачайте весы
   ```sh
   https://pjreddie.com/media/files/yolov3.weights
   ```
3. Запустите файл
   ```js
   python live_yolo_opencv.py
   ```
